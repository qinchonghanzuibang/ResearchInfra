# Code of Conduct

ResearchInfra follows the Contributor Covenant Code of Conduct.

## Our Standards

Participants are expected to be respectful, constructive, and welcoming. We do
not tolerate harassment, intimidation, or discriminatory behavior.

## Enforcement

Project maintainers may remove comments, issues, pull requests, or other
contributions that violate these standards. Serious or repeated violations may
lead to a temporary or permanent ban from project spaces.

## Reporting

For non-sensitive concerns, open a GitHub Issue that identifies the relevant
conduct concern and asks for maintainer review. For sensitive concerns, open a
minimal GitHub Issue requesting a private contact channel without including
personal or incident details. Maintainers should acknowledge the request and
move the discussion to a private channel as soon as practical.
