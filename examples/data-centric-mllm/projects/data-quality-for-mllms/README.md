# Data Quality for MLLMs

## Research Direction

Study how data quality, coverage, filtering, and evaluation design affect
multimodal language model behavior.

## Open Questions

- Which data-quality signals are inspectable before training or fine-tuning?
- How should claims about data coverage be linked to source audits?
- What baselines are needed before attributing improvements to data changes?

## Evidence Policy

Add papers under `sources/`, claims under `memory/claims/`, experiment plans
under `experiments/`, and concrete run records under `runs/`.
