# World-Model Pretraining

## Research Direction

Study how representation learning and predictive objectives can support
vision-language-action systems.

## Open Questions

- Which pretraining objectives produce evidence that transfers to downstream
  embodied tasks?
- How should run records distinguish representation probes from task-level
  evaluations?
- Which claims require simulator evidence, real-world evidence, or both?

## Evidence Policy

Keep claims separate from hypotheses until linked to papers, run records,
figures, tables, or documented evaluations.
